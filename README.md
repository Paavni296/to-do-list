# to-do-list
A simple To-Do List project made with HTML, CSS, and JavaScript.
## ✨ Features
- Add new tasks
- Mark tasks as done ✅
- Delete tasks ❌
- Task counter (done/total)
- Colorful & clean design

## 🚀 How to Run
1. Download or clone this repo.
2. Open `index.html` in your browser.
3. Start adding your tasks!
